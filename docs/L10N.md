# Luxora localization

Same **12 languages** as Discipline AI: `en`, `es`, `fr`, `de`, `zh`, `ko`, `ja`, `ru`, `it`, `hi`, `pt`, `tr`.

## Files

- Template: `lib/l10n/app_en.arb`
- Locales: `lib/l10n/app_*.arb`
- Generated: `lib/l10n/app_localizations.dart` (do not hand-edit)
- Config: `l10n.yaml`

## Workflow

1. Add or change keys in **`app_en.arb`** (and every other `app_*.arb`).
2. Regenerate:

```bash
flutter gen-l10n
```

3. Use in UI: `AppLocalizations.of(context).yourKey`

## Bootstrap from Discipline settings strings

To refresh shared settings labels from Discipline ARBs:

```bash
dart run tool/bootstrap_luxora_arb_locales.dart
flutter gen-l10n
```

## Settings UI

- **Concierge** → gear (top right) → Settings sheet (Discipline-style premium sheet)
- **Language** → same card picker as Discipline (flags + endonyms)
- **Appearance** → dark / light (persisted)
- **Countries & cities** → placeholder for future hub packs

Locale and theme are stored in SharedPreferences (`luxora_locale`, `luxora_dark_mode`).
